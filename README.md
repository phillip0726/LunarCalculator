# LunarCalendar(음력계산기)
- YYYY MM DD(양력)을 입력받아 naver 크롤링을 통해 음력 날짜를 추출합니다.

## version
- python : 3.7.0
- selenium : 3.141.0

## example
- input : `yyyy mm dd : 2019 3 24`
- output : 
&nbsp;```
*********Solar Calendar***********
Year  : 2019
Month : 3
Day   : 24

****Translate to Lunar Calendar***
Year  : 2019
Month : 2
;Day   : 18
```
